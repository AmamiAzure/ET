# 本项目包含多个解决方案

ET 7.0 + FGUI + luban + huatuo + YooAsset + NKGMoba，并提供常用的编辑器工具。

# 环境 && 版本

 - Unity：2020.3.33
 - 服务端：.Net 6.0
 - 客户端：.Net Framework 4.7.2
 - IDE：Rider 2022.1.2
